HTML:

It is a programing language that is used the create the base form of our website, here you will be taught everthing you need to know about html to get started with website development. HTML is mand up of all of tags think of tags like building blocks which you place to make something beautiful. All tags are either in pairs or single, tags are also classified into two main parts that is the opening and closing tags.



Opening Tags:

They are the first set in the pair of tags(if there is a pair) they are the only tags that can take attributes in which will be trated later. Opening tags also state where the tag starts and have a lot of importance in HTML.


Closing tags:

They are the second set of tags in a pair of tags the closing tag is also used to tell the computer to end a specific line of code. Just think of closing tags as the terminationg operator
